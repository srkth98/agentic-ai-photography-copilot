# Memory Cards & File Storage Guide

## Purpose

This guide covers memory card selection, file formats (RAW vs JPEG), backup
strategies, and troubleshooting storage-related issues.

## Memory Card Types and Speed

* **SD / SDHC / SDXC**: Common in most consumer and many professional cameras.
* **CFexpress / XQD**: High-speed cards used in professional bodies for high
  frame-rate burst shooting and high-resolution video.
* **Speed Class / UHS Rating**: Look for U3 or V30/V60/V90 ratings for 4K/8K video
  and high frame-rate burst shooting. Slower cards can cause the camera to "lock
  up" briefly while writing a large burst of photos.

## RAW vs JPEG

| Format | Pros | Cons |
|---|---|---|
| RAW | Maximum editing flexibility (exposure, white balance, highlights/shadows recovery) | Large file sizes, requires post-processing software |
| JPEG | Smaller files, ready to share immediately | Limited editing flexibility, compression artifacts if heavily edited |

For important shoots, shoot RAW (or RAW+JPEG) to preserve maximum editing
flexibility, especially for tricky lighting or white balance situations.

## Backup Strategy (3-2-1 Rule)

* **3** copies of your data
* On **2** different types of storage media (e.g. internal SSD + external HDD)
* With **1** copy stored off-site or in the cloud

For a shoot day: copy files from the card to a laptop/drive immediately, and to a
second drive before reformatting the card.

## Common Issue: "Card Full" or "No Card" Errors

### Symptoms

* Camera displays "card full" despite available space, or "no card" with a card
  inserted

### Resolution

1. Check that the card's write-protect switch (small physical slider on SD cards)
   is not in the "locked" position.
2. Reformat the card in-camera (after backing up all files) — files copied from a
   computer can sometimes leave a folder structure the camera doesn't recognize as
   having free space.
3. Try the card in a different camera or card reader to determine if the card
   itself or the camera's card slot is at fault.
4. Inspect the card's contacts for dirt or damage; clean gently with a dry cloth.
5. If the card consistently fails across devices, retire it — do not continue
   using a failing card for important shoots.

## Common Issue: Corrupted or Missing Files

### Symptoms

* Files show as 0 KB, won't open, or are missing after a shoot

### Resolution

1. Stop using the card immediately — do not take new photos or format it.
2. Use dedicated file-recovery software designed for the card's file system to
   attempt recovery before any further writes occur.
3. If recovery software finds files, copy them to a different drive immediately.
4. Going forward, always safely eject cards from card readers before removing
   them, and avoid removing the card or battery while the camera is writing to the
   card (indicated by the access light).

## File Organization Tips

* Use a consistent folder structure by date and shoot name
  (e.g. `2026-06-12_ClientName/`).
* Rename files with a consistent pattern including date and sequence number for
  easier sorting later.
* Tag or rate images during import to speed up culling in post-processing.

## Escalation Criteria

* If a card fails repeatedly across multiple devices, treat it as defective and
  replace it — do not rely on it for important shoots.
* If recovered files are still corrupted after using recovery software, a data
  recovery specialist may be able to help, but success is not guaranteed,
  especially if the card has been reformatted or written to again.
